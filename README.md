# deco

Rust-first dev container CLI.

Quick start:

```sh
cargo run -p deco -- --help
cargo install --path . --locked
deco --version
```

See `docs/installing.md` for install and local release commands.
See `docs/contributing.md` for the local quality gates that must pass before merge.
See `docs/stderr-convention.md` for the stderr logging contract and `crates/deco-core-model::diagnostics` for the shared helper.
